

# ===================== Emoji =====================
# ✨- :sparkles: 新機能の追加
# 🐛- :bug: バグフィックス
# ♻️- :recycle: リファクタリング
# 🔒️- :lock: セキュリティ改善
# ⚡️- :zap: 性能改善
# 🔥- :fire: 不要な機能/コード/ファイルの削除
# 🎨- :art: ファイル,ディレクトリ構造/コードフォーマットの改善
# 📝- :memo: ドキュメントの変更
# ✅- :white_check_mark: テストの追加/更新
# ⬆️- :arrow_up: 依存ライブラリのバージョンアップ
# ⬇️- :arrow_down: 依存ライブラリのバージョンダウン
# ➕- :heavy_plus_sign: 依存ライブラリの追加
# ➖- :heavy_minus_sign: 依存ライブラリの削除
# 💡- :bulb: ソースコード内のコメントの変更
# 🥅- :goal_net: エラーの解消
# 🎉- :tada: プロジェクトの作成
# 🔧- :wrench: 設定ファイルの変更
# 🚧- :construction: 後で修正または fixup される予定のコミット

# ===================== コミットメッセージの書き方 =====================
# :emoji: なぜどんな変更をしたのかを1行で要約。:emoji: は上記から選択
#
# - 必要に応じて箇条書きで変更点を書く
# - ただし具体的なファイルやクラス名を上げるわけではなく、
#   その修正によって何が変わったのかを書く
